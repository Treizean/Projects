Project 2: Classification Model on Yahoo Finance Dataset

This is a classification model on the stock market to predict whether a stock’s price will go up or down (or stay the same) based on historical data.

Detailed description: This project aims to build a machine-learning classification model to predict stock market movements (e.g., whether a stock’s price will go up or down). It involves collecting historical stock data (prices, volumes, and technical indicators), performing feature engineering (e.g., calculating moving averages, RSI), and preprocessing the data for modeling. A classification algorithm (e.g., Random Forest, XGBoost) is then trained to predict future price changes based on past patterns. The model's performance is evaluated using metrics like accuracy and precision, followed by backtesting to simulate real-world conditions. The goal is to create a predictive model that aids in stock market decision-making.

Project 3: Predicting Social Media Engagement: A Regression Analysis on Likes and Retweets

This analysis explores the use of linear regression to predict social media engagement metrics, focusing on likes and retweets, using sentiment, platform, and timing data.

Detailed description: This project investigates social media engagement factors by applying regression analysis to predict likes and retweets on posts. Using a dataset of social media posts with information on sentiment, platform (e.g., Twitter, Instagram), post timing, and associated hashtags, we analyzed the impact of these features on user engagement metrics. Linear regression models were built to identify trends and patterns in how different content attributes contribute to higher engagement levels. The model explained around 26% of the variance for both likes and retweets, indicating that while platform and sentiment play a role, other unobserved factors likely influence engagement outcomes. These findings suggest potential areas for model enhancement, including the incorporation of additional features, such as follower demographics or content quality indicators, to improve the predictive accuracy of social media engagement.
